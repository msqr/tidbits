TidBits: screen shots

Here are some screen shots that highlight some of the features of TidBits.

TidBits Home: This is what you see when you log into TidBits: a listing of all your tidbits of information. TidBits Types: Each tidbit of information is categorized as a type. You define what the types are, for example Password or Username.
Edit a tidbit: Creating or changing tidbits is easy. Each tidbit has a type, a name, and a tidbit of information (data). They can have an optional comment, too. Search for tidbits: To locate tidbits easily just search using the simple search form.
Search results: Search results are displayed just like the main tidbits view. Import CSV: You can import CSV data, i.e. to import data from a spreadsheet or database program.
Mobile device support: TidBits includes special support for mobile smart devices (such as the Palm Treo and iPhone). TidBits detects these devices and returns optimized HTML for faster access. This means it is very easy to access your TidBits database from virtually anywhere, which can come in handy if you're on the move. iPhone device support: TidBits works great on an iPhone.
